# CLAUDE.md — a11y-architect-demo

This project demonstrates the `/ecc:a11y-architect` agent — an Accessibility Architect specializing in WCAG 2.2 Level AA compliance.

## Purpose

Showcases accessible vs. inaccessible patterns side-by-side with WCAG 2.2 criteria annotations. Use this project to learn how the a11y-architect agent audits and fixes accessibility issues.

## When Editing

- Run `npx serve .` to preview the demo page locally
- Keep annotations clear — each fix should reference the specific WCAG 2.2 success criterion
- Maintain the before/after comparison pattern
- Use semantic HTML and ARIA attributes correctly

## Relevant Skills

- `/ecc:a11y-architect` — Accessibility Architect agent
- Skill: `accessibility` — WCAG 2.2 implementation guidance
